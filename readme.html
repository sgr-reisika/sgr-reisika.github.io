【改造について】

自分で使う用としての改造は、
お着替え、ボーン、物理演算、表情、カラー等々ご自由にどうぞ。
ただし、
  • ■著しくキャラクター性を損なう改造
  • ■該当モデルだとわからなくなる程の別キャラクターとしての改造
  • ■過度な露出をする等の猥褻表現にあたる衣装へのお着替え
  • ■同一キャラクターの別モデルを使用しての改造
  • ■製作者やベースがはっきりと分からない衣装やパーツなどを使用しての改造
この5つは禁止させていただきます。

例1：性転換、女装、年齢操作、極端な肥満体型、等→不可
例2：○○キャラモデルを△△風○○にしてみた→可
　 　○○キャラモデルの衣装を着た△△キャラモデルにしてみた→可
　　 ○○キャラモデルを△△にしてみた→不可
例3：SGR式○○モデルと☆☆式○○モデルでどうこう→不可


禁止事項における「パーツを流用しての他キャラクターへの改造」との兼ね合いがわかりづらいですが、
「○○キャラの衣装」という意味合いでの、
お着替え用衣装として他キャラモデルの顔と合わせる、というのは可です。
その場合、できるだけ「首から上」「首から下」という意味での顔、体、単位で組み合わせてください。
「○○キャラの衣装」としてのデザインやイメージなどを損なうような改造は不可です。


他モデル、又はお着替え用ボディとSGR式モデルを合わせる場合は、そちらのモデルの規約も守った上でご利用ください。
SGR式モデルを他モデルのお着替え用ボディとする場合にも、そちらのモデルの規約も守った上でご利用ください。

あとできればお着替えしたら体型など合わせてくれると嬉しい。
